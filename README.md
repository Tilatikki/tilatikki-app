# Tilatikki

Room reservation and freeing system

```sh
# Install all dependencies
npm run setup:project
# Start dev server
npm run dev:server
```

```sh
# Build for prod
npm run build
# Start prod server
npm start
```

## Tech use

- React
- Express L
- Redux
- React three fiber/drei
- Three.js
- TailwindCSS
- @Radix-UI
- Vitest
- TypeScript
- MongoDB
