```ts
git clone https://github.com/VirittamoHelsinki/tilatikki-app.git

cd tilatikki-app

npm run setup (Install all dependencies simultaneously for the front-end, back-end, and root).

npm run start-prod (To run the project's back-end and front-end simultaneously in a production state)
npm run start-dev  (To run the project's back-end and front-end simultaneously in a development state)


!!for working on backend remmember to run!!

npm run server-build (This will run the TypeScript compiler in watch mode, so every time you make any changes to files in the server folder, it will update the build folder automatically.)
```
